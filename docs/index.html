料金体系について

AutoBuildCraft で必要なコストはたった２つです。
ご安心ください、課金額以上のコストは発生しません。

  • 月額利用料（300円～）
  • サーバー利用料（チャージ）
サーバー利用料（チャージ）は、サーバーの稼働時間に比例してかかるコストです。
チャージは事前の購入が必要で100時間、1週間（168時間）、1ヶ月（720時間）分のチャージがあります。

チャージを購入後は忘れずにアクティベートしてくださいね！


サーバーに少しだけ制限があります😥

  • サーバー起動後、連続24時間程度経過するとサーバーが一度停止します
    • ※サーバーを起動しっぱなしでなければ影響はありませんよ
  • VPSなどとは違いMinecraftのサーバー機能のみを提供しています
